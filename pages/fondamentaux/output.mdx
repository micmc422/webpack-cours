# Output
L'output spécifie le nom et le chemin du fichier résultant généré par Webpack (le bundle). C'est le fichier que vous inclurez dans votre page HTML.

Exemple de configuration dans `webpack.config.js` :

```javascript copy
const path = require('path');

module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
  },
  // ...
};
```
## Exercice : Configurer l'output de Webpack

1. Modifiez votre fichier `webpack.config.js` pour changer le nom du fichier de sortie en `myBundle.js`.

2. Changez également le chemin de sortie pour qu'il pointe vers un nouveau dossier `dist/custom`.

3. Après avoir effectué ces modifications, exécutez à nouveau votre script de construction Webpack.

4. Vérifiez si le fichier `myBundle.js` est maintenant situé dans le dossier `dist/custom`.

Voici comment votre configuration devrait ressembler après les modifications :
```javascript copy
module.exports = { 
  // ... output: 
  { 
    filename: 'myBundle.js', 
    path: path.resolve(__dirname, 'dist/custom'),
    }, 
  // ... 
  };


```
