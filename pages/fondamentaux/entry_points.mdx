# Entry Points
L'entry point est le point de départ de votre application, le fichier JavaScript à partir duquel Webpack commencera à construire le graphe des dépendances. C'est généralement le point d'entrée principal de votre application.

Exemple de configuration dans `webpack.config.js` :

```javascript copy
module.exports = {
  entry: './src/index.js',
  // ...
};
```
## Exercice : Créer une application Webpack simple
<Steps>
### Structurer
Créez un nouveau dossier pour votre projet et naviguez vers ce dossier dans votre terminal.

### Initialisez
Initialisez un nouveau projet Node.js en exécutant la commande suivante

```bash copy
npm init -y
```

### Webpack et Webpack CLI
Installez Webpack et Webpack CLI en tant que dépendances de développement

```bash copy
npm install --save-dev webpack webpack-cli
```

### `index.html`
Créez un fichier `index.html` dans votre dossier de projet avec le contenu suivant

```html copy
<!DOCTYPE html>
<html lang="fr">
<head>
   <meta charset="UTF-8">
   <title>Webpack App</title>
</head>
<body>
   <h1>Bonjour de Webpack !</h1>
   <script src="dist/main.js"></script>
</body>
</html>
```

### `index.js`
Créez un fichier `index.js` dans un nouveau dossier `src` avec le contenu suivant :

```javascript copy
console.log('Bonjour de Webpack !');
```

### `webpack.config.js`
Créez un fichier `webpack.config.js` à la racine de votre projet avec le contenu suivant

```javascript copy
const path = require('path');

module.exports = {
 entry: './src/index.js',
 output: {
   filename: 'main.js',
   path: path.resolve(__dirname, 'dist'),
 },
};
```

### `package.json`
Ajoutez un script `build` à votre fichier `package.json` pour exécuter Webpack

```json copy
"scripts": {
 "build": "webpack"
}
```

### Exécutez le script `build`

```bash copy
npm run build
```

### Test
Ouvrez votre fichier `index.html` dans un navigateur et vérifiez si le message 'Bonjour de Webpack !' s'affiche dans la console.
</Steps>
